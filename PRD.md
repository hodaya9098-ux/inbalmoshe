# PRD — אתר עינבל מושה | Makeup & Nails

## מטרה
אתר תדמית לעינבל מושה, מאפרת ומעצבת ציפורניים עצמאית.
מאפשר ללקוחות לגלוש בגלריה, לבדוק מחירים, ולקבוע תור עצמאית.

## משתמש יעד
לקוחה חדשה או קיימת דרך הטלפון הנייד.

## דפים
- index.html — דף בית
- portfolio.html — גלריית עבודות
- price.html — מחשבון מחיר
- booking.html — קביעת תור (3 שלבים)
- admin.html — ניהול תורים

## טכנולוגיות
HTML, CSS, JavaScript, Firebase Realtime Database, GitHub Pages

## Definition of Done
- כל הדפים עובדים
- מחשבון מחיר פועל
- קביעת תור נשמרת ב-Firebase
- האתר תקין במובייל
